# SoraSi 🛍️
**A Modern E-Commerce Browser Application built with Flutter**

SoraSi is a fully functional e-commerce platform developed as part of my advanced Flutter training. The application focuses on providing a seamless user experience with dynamic data handling and smooth navigation.

---

## 🚀 Key Features
* **Dynamic Product Catalog:** Fetches and displays products from a REST API.
* **Smart Search:** Real-time text search to find specific products instantly.
* **Category Filtering:** Easily browse products using a custom Dropdown filtering system.
* **Product Details:** In-depth view for every item including high-resolution images and descriptions.
* **Cart & Favorites:** Integrated state management allowing users to save items and manage their shopping cart.
* **Responsive Design:** Optimized for browser use using Flutter's flexible layout widgets (Row, Column, Expanded).

---

## 🛠️ Technical Stack
* **Framework:** Flutter
* **Language:** Dart
* **State Management:** Stateful Widgets & Global State Logic
* **Navigation:** Flutter Navigator (Named Routes)
* **Networking:** `http` package for REST API integration
